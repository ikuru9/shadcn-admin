# Quick Start

## When to read

- When you want a quick checklist for the standard workflow

## Steps

1. Identify the task type and read the matching index guide(s)
2. Check whether the target path has a closer `AGENTS.md` and read it if present
3. Read only the detailed guide files needed for the task
4. Install dependencies: `pnpm install`
5. Start the dev server: `pnpm dev`
6. Make the code changes
7. Check linting: `pnpm lint`
8. Run tests: `pnpm test`
9. Recheck relevant restrictions before finishing
