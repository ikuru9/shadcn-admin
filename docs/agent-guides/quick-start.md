# Quick Start

## When to read

- When you want a quick checklist for the standard workflow

## Steps

1. Read only the guide files needed for the task
2. Install dependencies: `pnpm install`
3. Start the dev server: `pnpm dev`
4. Make the code changes
5. Check linting: `pnpm lint`
6. Run tests: `pnpm test`
7. Recheck relevant restrictions before finishing
