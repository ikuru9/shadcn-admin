# React Component Patterns

## When to read

- When building or refactoring component structure

## Rules

- Use function components and hooks
- Do not use `forwardRef` in React 19 code
- Use CVA when variants are needed
- Use `data-slot` attributes as styling hooks
- Use the Radix UI Slot pattern for composition when needed
- Pass refs as props instead of using `forwardRef`
- Use `React.ComponentProps<HTMLElement>`-based prop typing where appropriate
